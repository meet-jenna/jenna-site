agent.ts 
import { Agent } from "@jenna/agents";

const agent = await Agent.create({
  name: "Jenna",
  voice: "voice_jenna_warm",
  language: "en",
  tools: [takeOrder, bookReservation, sendToPOS],
  knowledge: ["./menu.pdf"],
});

agent.on("call", (call) => {
  call.say("Hi, you've reached Jenna — thanks for calling.");
});

Frequently asked questions

What is the conversational agents platform?
A managed runtime for voice agents — combining low-latency speech, language understanding, and tool use so you can ship production conversations fast.
How do I bring my own data?
Upload PDFs, connect URLs, or stream from your databases and CRMs. The agent answers grounded in the sources you authorize.
Which telephony providers are supported?
Native integrations with Twilio, Telnyx, SIP, and WebRTC. We also offer a browser SDK for in-app voice.
Is there a free tier?
Yes. You can build, test, and ship a small agent on the free plan. Upgrade when your traffic grows.
How do you handle security?
SOC 2 Type II, GDPR, HIPAA-aligned controls, regional data residency, and SSO/RBAC for enterprise teams.
